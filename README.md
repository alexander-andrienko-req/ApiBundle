ApiBundle
=============
The ApiBundle is abstract layer for creation of the RESTfull application

Requirements
------------
1) PHP >=7.4
2) Symfony >= 4.4.11

Installation
------------
Require the bundle with composer:

```bash
$ composer require requestum/api-bundle
```

Developing
------------
```bash
$ git clone git@github.com:requestum-team/ApiBundle.git
$ make install 
$ make test-unit
```
